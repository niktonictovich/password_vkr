# Сервис проверки надежности и генерации паролей

Веб-приложение для оценки надежности вводимого пользователем пароля и генерации криптостойких паролей с применением вероятностной модели машинного обучения (цепи Маркова, обученной на корпусе утекших паролей).

Проект разрабатывается в рамках выпускной квалификационной работы по направлению подготовки 10.03.01 «Информационная безопасность».

## Архитектура

Приложение реализовано в виде трех независимых микросервисов, запускаемых в Docker-контейнерах:

| Сервис | Назначение                                  | Технология | Порт |
|--------|---------------------------------------------|------------|------|
| `web`  | Пользовательский интерфейс                  | Streamlit  | 8501 |
| `api`  | Обработка запросов, оценка надежности       | FastAPI    | 8000 |
| `ml`   | Генерация паролей моделью на цепях Маркова  | FastAPI    | внутренний |

Сервисы изолированы друг от друга и обмениваются данными по HTTP в рамках внутренней Docker-сети. Оркестрация выполняется через `docker compose`.

## Используемые технологии

- **Python 3.11** — основной язык реализации
- **FastAPI** — веб-фреймворк для построения REST API
- **Streamlit** — фреймворк для быстрой сборки веб-интерфейса на Python
- **zxcvbn** — библиотека эвристической оценки надежности паролей (разработана Dropbox)
- **NumPy** — численные вычисления для модели на цепях Маркова
- **Docker, Docker Compose** — контейнеризация и оркестрация

## Запуск проекта

### Требования

- Docker Desktop с поддержкой WSL2 (для Windows) или Docker Engine (для Linux)
- Свободные порты: 8000 (API), 8501 (веб-интерфейс)

### Команды

Сборка и запуск всех сервисов:

```bash
docker compose up --build
```

После запуска веб-интерфейс будет доступен по адресу: http://localhost:8501

Остановка:

```bash
docker compose down
```
