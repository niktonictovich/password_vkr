# Документация по выпускной квалификационной работе

Эта папка — рабочее пространство для подготовки текста ВКР и финального оформления.

## Файлы

- **`development-log.md`** — лог разработки: что сделали, какие решения приняли, какие концепции и термины нужно включить в текст ВКР
- **`screenshots-checklist.md`** — список скриншотов: что уже снято, что еще нужно снять, в какую главу пойдет
- **`references.md`** — источники, на которые будем ссылаться (ГОСТы, статьи, документация). Заполняется по ходу.
- **`chapter-1-theory.md`** — черновик Главы 1 (теоретическая часть)
- **`chapter-2-analysis.md`** — черновик Главы 2 (обзор решений)
- **`chapter-3-development.md`** — черновик Главы 3 (разработка)

## Принципы

- Формат — Markdown. На финальном этапе переводим в Word + ГОСТ-форматирование.
- Записи делаем по ходу работы, а не задним числом.
- В `development-log.md` фиксируем не «что было сделано», а **«какие решения приняли и почему»** — это и есть будущий текст для Главы 3.
- Цитаты и формулировки, которые могут попасть в финальный текст, помечаем `[ВКР]` — потом будем выдергивать.
