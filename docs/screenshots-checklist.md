## Качество скриншотов

- **Windows + Shift + S** — выделение области экрана, результат в буфер
- Для скриншотов терминала — увеличить шрифт до читаемого
- На скриншоте видна вся ценная информация без скролла
- Для редакторов кода — лучше светлая тема (черный текст на белом печатается контрастнее)
- Для терминалов — темная тема нормально
- Единообразие в пределах главы: не смешивать темную и светлую тему

## Что НЕ нужно скриншотить

- Этот чат
- Сообщения об ошибках, которые мы потом исправили
- Промежуточные состояния, не несущие смысловой нагрузки
- Чужие проекты, открытые приложения, посторонние окна в кадре
